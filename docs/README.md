# Classificador SVM

## Criação da Base de dados
É criado diversos sinais de áudio com as palavras “um”, “dois” e “três”, representando as 3 classes da classificação.


## Extração de atributos

## Classificador multiclasses SVM
abordagem 1 vs 1 para a classificação multiclasses. Em caso de empate, a escolha da classe pode ser feita de forma aleatória.

## Otimização de hiperparâmetros
 validação cruzada K-fold
